Section 3 · Automation & AI Experience

Automation Systems I've Built

Real systems, in production, handling real data. Here are concrete examples of automation I've designed and deployed.

Example 1: JotForm ↔ Pipedrive Data Cross-Reference

Problem: A B2B SaaS company (GoBots) was running paid ads driving leads to JotForm, but 40%+ of leads were never making it into Pipedrive CRM — meaning sales was losing deals without knowing it.

What I built: An automated data cross-reference system that matched 291 JotForm leads against the Pipedrive database, identified missing records, flagged UTM attribution gaps, and generated an interactive HTML dashboard with channel-by-channel breakdown.

Result: Recovered lost leads, fixed the sync pipeline, and established UTM-to-revenue tracking that identified which ad channels actually drove qualified deals.

Example 2: Multi-Country Cold Email Automation

Problem: Needed to reach 3,184 B2B prospects across Mexico, Brazil, Argentina, and LATAM with personalized outreach — but zero manual work.

What I built: End-to-end pipeline: Apify scraper → n8n webhook → AI classification (GPT-4o-mini) → Supabase database with deduplication → country-segmented CSV exports (MX: 1,139 / BR: 1,809 / AR: 424) → 4-email automated sequences with AI-personalized copy.

Result: Fully autonomous prospect pipeline running weekly. From scraping to inbox in under 24 hours, segmented by country, industry, and lead score.

Example 3: CRM Pipeline Automation (HubSpot + Pipedrive)

What I built across multiple clients:

  • Auto-stage progression — leads move through pipeline stages based on behavior triggers, not manual updates
  • Lead scoring with n8n — custom workflows that score leads based on form data, email engagement, and page visits
  • Task auto-creation — when a lead reaches SQL status, the assigned rep gets a task with all context
  • Slack/WhatsApp notifications — real-time alerts when high-value leads enter the pipeline

Example 4: Automated Reporting Dashboards

What I built:

  • Looker Studio dashboards — live CRM data (Pipedrive + HubSpot) with daily refresh, showing MQLs, SQLs, pipeline value, conversion rates by channel
  • Ad performance reports — cross-platform (Meta + Google + LinkedIn) unified in a single dashboard with cost-per-stage metrics
  • Interactive HTML reports — custom-built data visualizations for stakeholder presentations (like the JotForm cross-reference above)

Why This Matters for Konko

These are the same types of systems Konko helps clinics automate — lead capture, data sync, qualification workflows, and reporting. I've built them from scratch for B2B SaaS. Applying this to healthcare clinic acquisition for Konko is a natural fit — I understand both the technical architecture AND the growth marketing strategy.
